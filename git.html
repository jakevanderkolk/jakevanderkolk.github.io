Using Git in the Humanities

What is Git and why should I use it?

Git is a Version Control System that can help you keep track of and coordinate your personal or collaborative projects. Originally designed for software development, Git is extremely flexible and very capable of dealing with humanities-related content. In particular, it can act as the following:

  • A Changes Tracker. Think of it as an "Undo" button that allows you to restore any previous version of a file or group of files.
  • A Synchronization Tool. Git allows you to download or upload a file or group files among multiple locations, e.g. your own personal desktop computer, a repository on GitHub, your laptop, a smartphone, or your office computer.
  • A Collaboration Coordinator. When working on a project with other people (or from different locations), Git tracks who does what and when, and tries to help resolve when contributions overlap.

These are just a few of the possible uses of Git. With a little experience, you can conceivably do even more with this tool, as many have already done.
